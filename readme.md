# 📚 English 504 Master — ۵۰۴ واژه ضروری انگلیسی

![English 504 Master](icons/icon-192.png)

**English 504 Master** یک اپلیکیشن وب پیشرو (PWA) برای یادگیری ۵۰۴ واژه ضروری زبان انگلیسی بر اساس کتاب معروف **"504 Absolutely Essential Words"** است. این برنامه با طراحی زیبا، تلفظ صوتی فارسی و انگلیسی، فلش‌کارت، تمرین روزانه و پیگیری پیشرفت، به شما کمک می‌کند تا واژگان انگلیسی را به‌صورت مؤثر و ماندگار یاد بگیرید.

---

## ✨ ویژگی‌های اصلی

- ✅ **۵۰۴ واژه کامل** از کتاب ۵۰۴ Absolutely Essential Words (۴۲ درس، هر درس ۱۲ واژه)
- 🎯 **تمرین روزانه** با ۶ واژه تصادفی از واژه‌های آموخته‌شده یا کل واژه‌ها
- 🔊 **تلفظ صوتی** انگلیسی و فارسی (با استفاده از Web Speech API)
- 📖 **فلش‌کارت** تعاملی با مثال‌های کاربردی و ترجمه فارسی
- 📝 **آزمون درس‌ها** با سوالات چهارگزینه‌ای و نمایش امتیاز
- 📈 **پیگیری پیشرفت** شامل:
  - تعداد واژه‌های آموخته‌شده
  - درصد پیشرفت کلی و هر درس
  - رکورد روزهای پیاپی (Streak)
  - تاریخچه تمرین‌های روزانه
  - بهترین امتیاز هر درس
- 🔍 **واژه‌نامه کامل** با قابلیت جستجوی فارسی/انگلیسی
- 🌐 **پشتیبانی آفلاین** با Service Worker (PWA)
- 📱 **طراحی واکنش‌گرا** و بهینه برای موبایل و تبلت
- 🎨 **پس‌زمینه روشن** و حرفه‌ای با رنگ‌بندی چشمنواز
